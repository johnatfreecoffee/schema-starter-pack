**EVERY PAGE YOU CREATE MUST HAVE:**

- ✓ Deep, professional shadows on ALL cards and buttons
- ✓ Rounded corners on EVERY element (minimum 12px border-radius)
- ✓ Smooth hover effects with transforms

- ✗ Sharp corners
- ✗ Cramped layouts
